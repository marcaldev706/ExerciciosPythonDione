## Descrição
Este repositório contém a resolução de **10 exercícios em Python**, desenvolvidos como atividade da disciplina de Técnicas de Programação.

Os exercícios abordam conceitos fundamentais da linguagem Python, como:

- Estruturas condicionais
- Estruturas de repetição
- Listas
- Filas
- Pilhas
- Entrada e saída de dados
- Lógica de programação
